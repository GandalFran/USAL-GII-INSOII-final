    Precondición Un cultivo escogido por el usuario esta presente en el sistema.
    Secuencia normal Paso Acción
    1 El sistema El sistema recupera la informacion del cultivo

    4 Si el usuario pulsa el boton de "Cancelar", el sistema indica al usuario mediante un mensaje que los datos no han sido borrados, a continuación este caso de uso queda sin efecto
