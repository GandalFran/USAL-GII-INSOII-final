2 Objetivos del sistema

OBJ-0004 Gestion de instalaciones, materiales y trabajos
Versión 1.0 ( 14/09/2018 )
Autores
  • Francisco Pinto Santos
Fuentes
  • Davinia Carolina Zato Dominguez
Descripción El sistema deberá gestionar las instalaciones, los materiales y los trabajos a realizar; concretamente todo lo relativo a la maquinaria, los terrenos, los silos, los trabajos y la el pienso.
Subobjetivos
  • [OBJ-0007] Gestion de maquinaria: El sistema deberá gestionar la maquinaria; incluyendo averias, presupuesto y coste de arreglos, y disponibilidad de las maquinas.
  • [OBJ-0008] Gestion de terrenos: El sistema deberá PD
  • [OBJ-0009] Gestion de silos: El sistema deberá PD
  • [OBJ-0010] Gestion de trabajos: El sistema deberá PD

Urgencia inmediatamente
