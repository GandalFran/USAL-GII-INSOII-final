UC-0013 Añadir cultivo
Versión 1.0 ( 15/09/2018 )
Autores
  • Francisco Pinto Santos
Fuentes
  • Davinia Carolina Zato Dominguez
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando o durante la realización de los siguientes casos de uso: [UC-0002] Listar terrenos no cultivados este año
Precondición El terreno seleccionado no tiene un cultivo registrado.
Secuencia normal Paso Acción
1 Se realiza el caso de uso Listar semillas (UC-0012)
2 Se realiza el caso de uso Recomendar cultivo y estimar produccion (UC-0009)
3 El sistema se le indica al usuario que escoga una semilla para cultivar entre las disponibles; recomendandole que cultive alguna de las indicadas en el paso anterior.
4 El actor Administrador (ACT-0001) escoge una semilla de las listadas
5 El sistema guarda la informacion necesaria para indicar que el terreno escogido ha sido cultivado
Postcondición El terreno queda registrado como que ha sido cultivado.
Excepciones Paso Acción
1 Si no hay semillas disponibles, el sistema indicara que no hay semillas disponibles, a continuación este caso de uso queda sin efecto
4 Si el actor Administrador pulsa el boton de "cancelar", el sistema cancela el proceso de añadir cultivo, a continuación este caso de uso queda sin efecto
Rendimiento Paso Tiempo máximo
- -

UC-0009 Recomendar cultivo y estimar produccion
Versión 1.0 ( 15/09/2018 )

  • Francisco Pinto Santos
Fuentes
  • Davinia Carolina Zato Dominguez
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando o durante la realización de los siguientes casos de uso: [UC-0013] Añadir cultivo
Precondición -
Secuencia normal Paso Acción
1 Se realiza el caso de uso Ver detalle terreno (UC-0011)
2 Se realiza el caso de uso Listar historial de cultivos de un terreno (UC-0004)
3 El sistema realiza los calculos pertinentes con los datos obtenidos en los pasos anteriores sobre los cultivos recomenados para un terreno y los datos de produccion de años anteriores.
4 El sistema muestra una lista con los cultivos recomendados y los datos de produccion estimada para cada uno.
Postcondición Se muestra el cultivo recomendado y la produccion estimada
Excepciones Paso Acción
1 Si no se encuentran datos sobre el terreno, el sistema indicara que no se pueden realizar estimaciones por falta de datos sobre el terreno, a continuación este caso de uso queda sin efecto
2 Si actualmente no hay registros sobre cultivos sobre el terreno escogido, el sistema se indica que no se van a poder realizar calculos estadisiticos sobre la produccion pero que si se indicaran los cultivos recomendados, a continuación este caso de uso continúa
